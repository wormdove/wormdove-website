- Capture the energy and setup
- Include lighting/visual elements if present

## üîß **File Optimization Tips**

### **Music Files:**
- **MP3**: 320kbps for final releases, 128kbps for previews
- **WAV**: Use for highest quality, but larger file sizes  
- **FLAC**: Lossless compression, good middle ground
- Keep preview clips to 1-2 minutes for web

### **Audiovisual Files:**
- **MP4**: H.264 codec, 1080p max resolution
- **File Size**: Keep under 100MB for web compatibility
- **Length**: 2-5 minutes ideal for portfolio pieces
- **Compression**: Balance quality with loading speed

### **Live Performance Videos:**
- Document key moments, not entire sets
- Include both wide shots and close-ups of equipment
- Good audio quality is essential
- 3-8 minute highlights work best
